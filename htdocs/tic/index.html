A alocação dinâmica de memória é a base das estruturas dinâmicas de dados. Uma estrutura dinâmica de dados é, tipicamente um conjunto de nós (nodes) ligados por um conjunto de arcos direccionados (directed edge) entre esses nós. Os nós contêm a informação e os arcos são ligações funcionais entre os nós que permitem percorrer a estrutura de dados. Em C, ao conceito de arco direccionado está associado o conceito de ponteiro (um ponteiro, no fundo, pode ser visto como uma ligação direccionada entre dois endereços de memória: o endereço onde o ponteiro reside e o endereço por ele apontado).

Tipos de Estruturas


Como foi dito anteriormente, a natureza das ligações entre os nós define as características de uma estrutura de dados. As estruturas de dados podem ser divididas em dois grandes tipos:

  • Lineares: cada nó está ligado apenas ao anterior e ao seguinte, criando uma estrutura "em linha". São exemplos deste tipo de estruturas as listas ligadas (filas, pilhas) unidireccionais ou duplamente ligadas.

  • Não lineares: cada nó está ligado a vários nós, criando uma estrutura mais ou menos arbitrária de nós interligados. Os exemplos mais importantes deste tipo de estruturas são as árvores e os grafos.
