Comandos de Cmd



dir - Este comando é usado para exibir uma lista de arquivos e pastas no diretório atual. Por padrão, ele exibe os nomes, a data e hora de criação e o tamanho dos arquivos. Você pode usar a opção /p para exibir os arquivos uma página de cada vez e a opção /w para exibir os arquivos no formato largo.

cd - O comando "mudar diretório" é usado para navegar pelo sistema de arquivos. Por exemplo, cd c:\ mudará o diretório atual para a raiz da unidade C. Você também pode usar caminhos relativos para navegar para subdiretórios, por exemplo, cd myfolder mudará para a pasta chamada "myfolder" se ela existir no diretório atual.

mkdir - Este comando é usado para criar um novo diretório. Por exemplo, mkdir myfolder criará uma nova pasta chamada "myfolder" no diretório atual.

rmdir - Este comando é usado para excluir um diretório vazio. Por exemplo, rmdir myfolder excluirá a pasta chamada "myfolder" se ela existir no diretório atual e estiver vazia. Para excluir uma pasta com arquivos, você pode usar o comando rd em vez disso.

del - Este comando é usado para excluir um arquivo. Por exemplo, del myfile.txt excluirá o arquivo chamado "myfile.txt" se ele existir no diretório atual. Você também pode usar curingas, como del * para excluir todos os arquivos no diretório atual.

copy - Este comando é usado para copiar um arquivo. Por exemplo, copy myfile.txt myfile_copy.txt criará uma cópia do arquivo chamado "myfile.txt" com o nome "myfile_copy.txt" no diretório atual.

XCOPY - O comando XCOPY no CMD é usado para copiar arquivos e pastas de um local para outro, incluindo sub-pastas e arquivos ocultos.

"tracert" - mostra o caminho de pacotes de rede até o destino

"ipconfig" - exibe informações de configuração de rede do computador

"netstat" - exibe informações sobre conexões de rede

"net" - gerenciamento de usuários, grupos e compartilhamentos de rede

"systeminfo" - exibe informações gerais sobre o sistema

"driverquery" - exibe informações sobre os drivers instalados

"shutdown" - desliga o computador.

"cmd" - abre uma nova janela do prompt de comando

"ren" - renomeia um arquivo ou pasta

"type" - exibe o conteúdo de um arquivo de texto

"ping" - verifica a conectividade de rede com outro dispositivo

"date" - exibe a data atual

"time" - exibe a hora atual

"ver" - exibe a versão do sistema operacional

"vol" - exibe o número de série do volume

"chkdsk" - verifica e conserta erros no disco
